The SYSUTILS plugin now supports the "SYSUTILS_LOAD" environment variable.

If, at load time, this variable is set to a comma-separated list of

commands, functions, and variables exported by SYSUTILS.DLL, only the

specified items will be imported by 4NT (see example below).


This mechanism is not forgiving; specifying items not exported by SYSUTILS.DLL

will result in errors when 4NT tries to import them.


This mechanism is especially intended for use in enduring batch files

where the speed of execution is of importance.  A batch file using only

a few items might use, for example, 


       plugin /U *

       set sysutils_load=@connect,uptime

       plugin /L [path\to\]sysutils.dll

       

Later, to reload the default (auto-load) plugins, such a batch file

might use


       plugin /U sysutils

       unset sysutils_load

       plugin /L *

  string: case-insensitive unqualified module name substring match criterion


  Format elements (lowercase suppresses description; default: IQBRvUKTWVPOC)


    I - PID            Q - File (qualified)    F - File             O - Owner

    B - Begin          R[s|m|h|d|v] - Running  U - User CPU         N - newline

    K - Kernel CPU     T - Total CPU           P - Priority class

SETPRIORITY


Set process priority class


  SETPRIORITY process_id [ I | B | N | A | H | R ]


  Options (only the first character is significant)

          IDLE, BELOW_NORMAL, NORMAL, ABOVE_NORMAL, HIGH, REALTIME


  REALTIME will pre-empt normal system activity; use it cautiously.


  Given only process_id, SETPRIORITY will print the process's priority class.

  Also see @PRIORITY.

  /C          use context menu


  /V verb     "print", "edit", "explore" et c.

              (with /C) "properties", "preview", et c.  Without "/V verb",

              the default verb is used if available, else "open" is used.


  /D          (debug) show parameter parsing without executing


  <arguments> and <directory> apply to launching executables


  When appropriate, <file> may specify a URL, a directory or a drive


  Quote parameters containing whitespace; escape necessary quotes as \"

       /C        wait for window CREATE, default: wait for SHOW)

       /D        wait for window DESTROY

       /MS|/ME        wait for move-size start/end

       /NC        wait for new caption

               T(itle)   V(isible)  S(how)    L(TRB normal)

  /E | /D        enabled | disabled windows only

  /V | /I        visible | invisible windows only

  /P pid        decimal or hex prefixed with "0x"

  /M module        case-insensitive substring match on fully-qualified name

  /C class        case-insensitive substring match

  /T title        case-insensitive substring match

               "quote" strings containing spaces
