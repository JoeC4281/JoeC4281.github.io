RENCASE — Change the case of filenames.

Syntax:
RENCASE /A:attribs /E /L /N /N:flags /S /U /W /X:luw /[range] file…

/A:attribs

filter by attributes

/E

no error messages

/L

change the base name (all but the extension) to lowercase

/N

not really

/N:flags

D no hidden directories, J no junctions, Z no system directories

/S

look in subdirectories for matching files

/U

change the base name (all but the extension) to uppercase

/W

change the base name (all but the extension) to word case

/X:luw

change the extension to upper/lower/word case

/[range]

many range options are supported

file…

wildcards, directory aliases, etc. supported

RENCASE renames files, changing the case of letters. Filenames are handled as two parts: the base name (everything before the last period), and the extension (everything after). Options /L, /U, and /W affect the base name, and /X: affects the extension.

If no action is specified (none of /L, /U, /W, or /X:), the default is /L.

rencase /l /x:l *.html
