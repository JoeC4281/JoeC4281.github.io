Copies text from the clipboard to a file or standard output.

Syntax:
CLIP2TEXT /A /NB /O /P /T /UTF8 /UTF16 filename

/A

append to an existing file

/NB

do not write a Byte Order Mark

/O

overwrite an existing file

/P

page output (useful only when output is to stdout)

/T

quietly

/UTF8

write file in the UTF-8 encoding

/UTF16

write file in the UTF-16 encoding (default)

Only one filename is allowed. If no filename is specified, CLIP2TEXT will dump the clipboard to standard output.

See also: the TEXT2CLIP command.
