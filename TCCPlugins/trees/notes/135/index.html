PSTAT


Display process data


PSTAT [ pid | exename | * ]  [ /F format ]


  string: case-insensitive with ".exe"


  Format elements (lowercase suppresses description; default: IQBRvUKTT+YY+WVPOCN)


    I[+] - ID [parent] Q - File (qualified)    F - File             O - Owner

    B - Begin          R[s|m|h|d|v] - Running  U - User CPU         N - newline

    K - Kernel CPU     T[+] - Total CPU [%]    P - Priority class   Y[+] - cycles [%]

    W - Working Set    V - Virtual Memory      C - Command line


  CPU times in seconds; R defaults to Rs; memory in KB
