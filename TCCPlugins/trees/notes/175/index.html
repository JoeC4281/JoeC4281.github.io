@ORD — Returns a number with an English ordinal suffix.

Syntax:
%@ORD[n]

n

a non-negative integer value

C:\> for /l %n in (1,1,5) echo %@ord[%n]
1st
2nd
3rd
4th
5th

Commas will be inserted for values over 999.

• New: This function does not exist in Take Command.
