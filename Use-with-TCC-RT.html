Use Libraries With TCC-RT
This tip presumes that you are using both TCCxx and TCC-RTxx.

I've created a TCMD.INI file, and placed it in my TCC-RTxx folder.

The TCMD.INI file contains the following;

[4NT]
LibraryDirectory=C:\Program Files\JPSoft\TCMD28\library

From the Help File:
When TCC starts, it will automatically load any library function files in the LIBRARY subdirectory of the TCC installation directory. You can specify a different location by setting the LibraryDirectory INI directive.
