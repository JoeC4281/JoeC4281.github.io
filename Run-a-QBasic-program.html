Run a QBasic program

On my Windows 10 64-bit system, I am able to run QBasic programs
via the MS-DOS Player for Win32-x64 console

The MS-DOS Player for Win32-X64 can be downloaded from
http://takeda-toshiya.my.coocan.jp/

set qbasic=E:\DOSBox\utils\qbasic.exe
msdos.exe %qbasic

start "msdos" /inv /pgm msdos.exe %qbasic /run r:\test.bas
