Get A List of all COM Servers

NOTE: The code below should be entered as just one line.

 

pshell /s "dir REGISTRY::HKEY_CLASSES_ROOT\CLSID -include PROGID -recurse | foreach {$_.GetValue('')}"

 

The above returns the list of all 64-bit COM Servers on my system.

 

pshell /s "dir REGISTRY::HKEY_CLASSES_ROOT\WOW6432Node\CLSID -include PROGID -recurse | foreach {$_.GetValue('')}"

 

The above returns the list of all 32-bit COM Servers on my system.

 

I have made the above as two seperate aliases;

 

  • progids64
  • progids32

 

The results are returned un-sorted, so I pipe the output through SORT;

 

progids64 | sort

progids32 | sort
