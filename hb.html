hb.library for use with Take Command Console
hb.library$hb {
@setlocal
@echo off
::
:: hb date()
:: hb 2022-1960
::
iff %1 eq help .or. %# eq 0 then
echo USAGE: hb date()
echo        hb 2022-1960
quit
endiff
:: Disable the Escape Character
setdos /X-8
if exist r:\ (set dest=r:\hb.prg) else (set dest=%temp\hb.prg)
echo Proc Main() > %dest
echo ?? %* >> %dest
echo Return >> %dest
set _hb=%@execstr[e:\utils\hbrun.exe %dest]
echo %_hb
setdos /X+8
endlocal _hb
}
NOTES: Drive R: on my system is a RAMDrive

This library command uses the external hbmk2.exe, for which I have created the following Symlinks;

2021-04-18   9:00     <SYMLINK>    hbrun.exe [e:\hb32\bin\hbmk2.exe]
2021-04-18   9:01     <SYMLINK>    hbrund.exe [e:\hb32\bin\hbmk2.exe]

This library command returns the result in the environment variable _hb

Thus, hb ?? 30*12 will store the result in _hb.

You can then echo %_hb to see the result.
