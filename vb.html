vb.library for use with Take Command Console
vb {
@setlocal
@echo off
:: Disable the Escape Character
setdos /X-8
if %1 eq help (
  echo vb 140.97*52
  echo vb dateadd("m", 708, "25-Nov-1960")
  echo vb DateDiff("d", "11/25/1960", Now) / 365.25
  quit
)

if exist r:\ (set dest=r:\vb.vbs) else (set dest=%temp\vb.vbs)

>%dest echo result=%*
>>%dest echo TakeCommand.tcommand("echo " + CStr(result))

set _vb=%@execstr[script %dest]

endlocal _vb
}
NOTES: Drive R: on my system is a RAMDrive

This library command uses the internal TCC commands
- TakeCommand.tcommand
- Script

This library command returns the result in the environment variable _vb

Thus, vb 30*12 will store the result in _vb.

You can then echo %_vb to see the result.
