xb.library for use with Take Command Console
xb {
@setlocal
@echo off
::
:: xb date()
:: xb 2022-1962
::
iff %1 eq help .or. %# eq 0 then
echo USAGE: xb date()
echo        xb 2022-1960
quit
endiff
:: Disable the Escape Character
setdos /X-8
if exist r:\ (set dest=r:\xb.prg) else (set dest=%temp\xb.prg)
echo Proc Main() > %dest
echo OutStd(%*) >> %dest
echo Return >> %dest
set _xb=%@execstr[E:\xharbour\bin\xbscript.exe %dest]
echo %_xb
setdos /X+8
endlocal _xb
}

NOTES: Drive R: on my system is a RAMDrive

This library command uses the external xbscript.exe.

This library command returns the result in the environment variable _xb

Thus, xb 30*12 will store the result in _xb.

You can then echo %_xb to see the result.
