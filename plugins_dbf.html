dbf Plugin for Take Command / TCC / TCC/LE

Beta Version 2021.10 Build 8

Joe Caverly

Purpose:

This plugin implements various functions to allow the reading of a dBASE .dbf file.

Installation:

To use this plugin, copy dbf64.dll to some known location on your hard drive. Load the plugin with a PLUGIN /L command, for example:

plugin /l e:\utils\dbf64.dll

If you copy these files to a subdirectory named PlugIns within your Take Command program directory, the plugin will be loaded automatically when TCC starts.

Plugin Features

Functions:
@BytesInHeader Returns the number of bytes in the header of the .dbf file.
@DBFVersion Returns the version of the .dbf file.
@FieldCount Returns the number of fields in the .dbf file.
@FieldDecimal Returns the number of decimal places of a numeric field.
@FieldName Returns the field name of a field, given the field number.
@FieldType Returns the field type of a field, given the field number.
@FieldWidth Returns the field width of a field, given the field number.
@FirstRecOffset Returns the byte in the file where the first record begins.
@LUpdate Returns the date that the .dbf file was last updated.
@RecNum Returns the record number, given the byte position in the file.
@Record Returns the contents of a record, given the record number.
@RecCount Returns the number of records in the .dbf file.
@Recsize Returns the size of a record in the .dbf file.
