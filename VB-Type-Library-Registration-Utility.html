VB Type Library Registration Utility

Ref: Gitub Project Site

The name of the program executable is VBRegTLB6.exe

Whilst all systems are provided with the regsvr32 tool for register ActiveX controls and DLLs, there is not an equivalent tool available for Type Libraries.

This page provides a Type Library registration utility that allows you to register Type Libraries more reliably than using VB's Project->References dialog.

It also allows you to unregister Type Libraries and browse through all the type libraries installed on the system, including deleting entries for old and deleted items.
