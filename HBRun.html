To debug a .prg using HBRun;
Modify your .prg source file, to include the following, at the location in your .prg source file, at which you want to start debugging;

AltD( 1 )                // activates debugger
AltD()                   // invokes debugger unconditionally

To begin debugging your .prg source file;

hbrun myfile.prg --hb:debug
I cannot get the screen size correct when doing this from a TCC window in TCMD.

Thus, I detach a TCC window from TCMD, and use MODE 80,25 to set the screen size, before starting hbrun.

@setlocal
@echo off
set lines=%@execstr[3,mode con:]
set lines=%@word[-0,%lines]
if %lines ne 25 mode 80,25
set columns=%@execstr[4,mode con:]
set columns=%@word[-0,%columns]
if %columns ne 80 mode 80,25
if %# gt 0 hbrun.exe %1 %2 %3 %4 --hb:debug
endlocal
